Welcome

This is an unofficial, unaffiliated app for displaying game information for players of the Star Trek Adventures game by Modiphius® Entertainment (links: US / UK).

  • department colors border (upper-left hidden selector)
  • pips for rank
  • current and maximum stress
  • name
  • photo (drag-and-drop an image or use the settings dialog)
Keyboard Shortcuts
Ctrl + S Save Changes
Ctrl + E Export Game to File
Ctrl + I Import Game from File
Ctrl + Comma (,) Open Settings
F1 Show Welcome Screen
Escape Hide any dialog

The starship's 3D model (.gltf/.glb) and player photos can be changed either via drag-and-drop or settings dialog.

This application is free and open source software.
